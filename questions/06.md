# Sinatra: Routing

A **route** consists of two parts: the [part1] (which specifies
the type of action being requested) and the [part2] (also known as the
resource).

## Correct

- HTTP verb
- part1

## Correct

- path
- part2

## Incorrect

- headers
- status code
- client
- request
